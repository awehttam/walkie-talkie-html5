JavaScript Integration

You can also communicate with the embedded walkie talkie using JavaScript:

// Listen for messages from the embedded walkie talkie
window.addEventListener('message', function(event) {
    if (event.data.type === 'walkie-talkie-connected') {
        console.log('Walkie talkie connected to channel:', event.data.channel);
    }

    if (event.data.type === 'walkie-talkie-speaking') {
        console.log('Someone is speaking:', event.data.speaking);
    }
});

// Send messages to the embedded walkie talkie
function sendMessageToEmbed(action) {
    const iframe = document.getElementById('embed-default');
    iframe.contentWindow.postMessage({
        type: action
    }, '*');
}

Installation Instructions

  1. Make sure the WebSocket server is running: php server.php
  2. Serve the web files from the public/ directory
  3. Update the WebSocket URL in the embed if needed
  4. Test microphone permissions (requires HTTPS in production)
  5. For notifications: Allow notification permissions when prompted
